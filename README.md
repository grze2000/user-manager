# User Manager

Discord bot returning filtered list of users in a guild

## Environment variables

Example .env file:

```
TOKEN=""
APPLICATION_ID=""
PREFIX=!
DEBUG=false
THUMBNAIL_URL=""
USE_EMBED=false
REQUIRE_ADMIN=true
MONGO_URI=""
TESTER_ID=""
TEST_GUILD_ID=""
```

## Commands

Install packages:
`pnpm i`

Run in watch mode for development:
`pnpm start:dev`

Compile typescript for production:
`pnpm build`

Run compiled version:
`pnpm start:build`
